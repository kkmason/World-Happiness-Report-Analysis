Technical Description

Data Collection

Data was collected using bulk data and web scraping. Bulk data was accessed from the World Happiness Report and the Gross National Happiness Index.

Because most data is proprietary, I was limited to publicly available data. Using a webscraping script, the ranking of countries was obtained from the wikipedia 2016 country rankings listed on the The World Happiness Report

Data Analysis Script

The data analysis script aggregates countries by continent to show how countries compare and longitudinal patterns. Additionally, breakdown by country shows how they compare.
